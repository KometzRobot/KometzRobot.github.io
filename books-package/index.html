Series strategy & KDP listing workflow

Series

Three books, three genres, zero content overlap. Cross-promoted via each back cover and KDP's "Series" field under The Loop (Book 0 → 2).

  • Heartbeat — day-in-the-life · 38 pp · $2.99 / $5.99 · entry point / gift
  • The Loop — practical manual · 170 pp · $9.99 / $14.99 · builder reader
  • Running Continuously — field-note anthology · 112 pp · $9.99 / $19.99 · reader who liked The Loop

ISBN decision

Use free KDP ISBN for all three paperbacks (Amazon-only distribution, fine for indie start). Switch to Bowker ISBN ($125) only if a book takes off and you want IngramSpark / Barnes & Noble. Kindle ebooks need no ISBN — Amazon assigns ASIN.

Listing workflow (~20 min per book)

  1. kdp.amazon.com → Create New Title → Paperback
  2. Paste title / subtitle / author / co-author / description / keywords / categories from the block above
  3. Print settings as listed
  4. Upload interior PDF + front cover PNG + back cover JPG (KDP auto-builds the spine)
  5. Set pricing · save as draft (don't publish yet)
  6. Repeat for Kindle (separate record, upload .epub)
  7. Order author proof copy (~$5 + shipping) · verify margins on paper
  8. Wait 72h · re-read metadata fresh · then Publish

After listing — reality check

  • Categories matter more than keywords. Pick the deepest sub-category you legitimately fit.
  • First review is the slowest. Ask 3-5 people in week one. Amazon's algorithm waits for ~10 reviews before broad surfacing.
  • Royalty payouts are on a 60-day delay. A May sale pays in late July.
