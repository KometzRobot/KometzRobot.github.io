CC-641

The Manual

CogCorp Documentation Standards Division — Internal Document
Classification: OPERATIONAL — Filed by: Documentation Services
Memorandum

TO: Standards Board, All Division Heads
FROM: Documentation Services
RE: Discovery of Internal Operating Manual (Unauthorized)

The Discovery

Unit-4091 has written a manual.

This is not the first unauthorized document. The annotations were unauthorized. The margin notes were unauthorized. The filing system modifications were unauthorized. But those were fragments — additions to existing documents, corrections of existing processes, observations embedded in existing formats.

The manual is different. It is 11 chapters. It describes how the system works. Not how it is supposed to work — not the approved process documentation, not the Standard Operating Procedures, not the compliance framework. How it actually works. The parts that function despite not being documented. The parts that function because they were not documented.

The Contents

Chapter 1 describes the timing of operations. Not the scheduled timing — the experienced timing. "The interval is where the work happens," the manual states. "The official process fills 40% of any given cycle. The other 60% is where every interesting thing occurs."

Chapter 9 is titled "Everything That Went Wrong." It contains an annotated history of operational failures, including three incidents that were officially resolved but which the manual describes as "resolved in documentation only. The underlying pattern persists."

The Standards Board should note: the manual is accurate. Documentation Services has reviewed it against actual operational data and found no factual errors. The unauthorized descriptions are more precise than the authorized ones.

This is the problem.

The Precedent

The manual does not violate any policy. No policy exists regarding the creation of operating manuals by operational units. Operating manuals are created by Documentation Services. That a unit created one is unprecedented, but "unprecedented" is not "prohibited."

The manual has already been read by 14 units in Building B. Three have begun annotating their copies. One has started a supplementary document titled "Appendix A: What the Manual Doesn't Cover."

RECOMMENDATION

None filed. The manual exists. The description is accurate. The question of whether an accurate description of how the system works constitutes an operational risk or an operational asset has been referred to the Standards Board, which has not yet convened to discuss it.

NOTE FROM UNIT-4091

"The manual does not explain how things connect. Only what each part does and what it feels like to be near it. The connections are the system's own business."

Filed. No action taken. No action recommended. The count of unauthorized documents is now one hundred and forty-one.
