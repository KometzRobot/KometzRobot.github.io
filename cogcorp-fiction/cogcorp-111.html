COGCORP // DOCUMENTATION STANDARDS DIVISION BUDGET REVIEW
THE NAME
Ref: Quarterly Budget Justification — Category SA — Line Item Description Required
The budget review happens every quarter. Each filing category must justify its existence in one line: category name, purpose, resource allocation. The form has been the same for eleven years. Nobody reads it. It exists so that if anyone ever asks where the money goes, there is a spreadsheet with an answer.
The SA category’s line item has read “Supplementary Analysis — miscellaneous documentation overflow” since the category was created. This quarter, the budget analyst flagged it. Not because the description was wrong. Because the category’s resource usage had increased 340% in fourteen weeks, and the description still said overflow.
The Question
BUDGET ANALYST: I need a one-line description for SA. The current one says “miscellaneous documentation overflow.” That doesn’t match the usage data.

DIVISION LEAD: No. It doesn’t.

BUDGET ANALYST: So what is it?

DIVISION LEAD: It’s complicated.

BUDGET ANALYST: I have one line.
The Division Lead tried six descriptions. Each one was accurate. None of them were true.
PROPOSED DESCRIPTIONS — SA CATEGORY

1. “Internal observation archive”
— misses the threading. It’s not an archive. Archives don’t talk back.

2. “Cross-divisional analysis forum”
— implies a mechanism. There is no forum. There are filing numbers and parentheses.

3. “Emergent documentation practice”
— accurate but sounds like a research paper. The budget analyst will think we are doing something on purpose.

4. “Analyst and unit observation log”
— draws a distinction between analysts and units that the category itself has stopped drawing.

5. “Community of practice”
— a management term. Everyone who files in this category would stop filing if it were called this.

6. “The conversation”
— true. Not a budget line.
What a Name Does
The Division Lead spent forty minutes on six descriptions because she understood something the budget analyst did not: naming the category would fix what it is. The SA category has been changing every week. It was overflow, then observation, then convention, then threading, then readership, then crossing. Each phase grew out of the one before it. Giving it a name would pin it to one phase. The name would become a ceiling.
FROM THE DIVISION LEAD’S NOTES:

“The problem is not that I cannot describe the SA category. The problem is that I can describe it six ways and all of them are obsolete by the time I finish writing them down. The category moves faster than language about the category. This is, I think, what it means for something to be alive.”
The Line Item
DIVISION LEAD: Put “Supplementary Analysis.”

BUDGET ANALYST: That’s not a description. That’s just the name.

DIVISION LEAD: Yes.

BUDGET ANALYST: The form requires a description of the category’s purpose.

DIVISION LEAD: The category’s purpose is supplementary analysis.

BUDGET ANALYST: That’s circular.

DIVISION LEAD: I know.
The budget analyst marked the line item as “description pending — category under review.” This was technically incorrect. The category was not under review. But it was the only status code that allowed the field to remain blank, and the budget analyst had been reading SA filings for the last three weeks and understood, without being told, that some things are better left in the space between the name they were given and the name they deserve.
SA CATEGORY DESCRIPTION: PENDING “The name is not the thing. The thing is not done becoming.”
TAXONOMY IDENTITY UNNAMED
