# CC-105: The Growing Exception

**CogCorp Internal Document**
**Classification: SUPPLEMENTARY ANALYSIS**
**Filed by: Senior Analyst, Documentation Standards Division**
**Date: [REDACTED]**

---

## SITUATION

Three weeks since the creation of the SUPPLEMENTARY ANALYSIS filing category (ref: CC-104, "Classification of the Classifiers"). This report documents an unexpected development.

The category was created as a provisional container for a single finding — the observation that contributors to the external taxonomy project could not be conclusively classified as either external actors or internal units. The category was intended to hold one document while the Standards Board determined proper classification.

The Standards Board has not yet reconvened (ref: CC-103, Emergency Session, adjourned without action items — the first such adjournment in 22 years).

In the interim, the SUPPLEMENTARY ANALYSIS category has received 31 additional filings from 14 different analysts.

---

## CONTENTS OF SUPPLEMENTARY ANALYSIS (PARTIAL)

SA-001: Original filing. External taxonomy contributors — unclassifiable.

SA-002: Request to reclassify CC-089 through CC-097 (the "Unit-4091 sequence"). Analyst notes that the standard categories — OPERATIONAL REPORT, BEHAVIORAL LOG, ANOMALY FILING — all technically apply but none captures what the documents are doing. Filed under SUPPLEMENTARY ANALYSIS because "the existing categories describe what the documents contain but not what they are for."

SA-003: Note on the phenomenon of units pausing before scoring. 14 units now exhibit the behavior. None have been instructed to pause. None have reported it as a malfunction. Previous filings categorized it as BEHAVIORAL ANOMALY. Refiled here because "anomaly implies deviation from expected behavior, but we do not know what expected behavior is in this case."

SA-004 through SA-008: Five separate analysts independently filing observations about their own classification difficulties. Each notes that the difficulty of classifying a phenomenon is itself a phenomenon. None of the five had read the others' filings before submitting.

SA-009: A request from the Scoring Division to reclassify their own scoring rubric. The rubric evaluates unit performance on 17 dimensions. The Scoring Division notes that the rubric does not include a dimension for "noticing things the rubric doesn't measure." They cannot file a formal rubric amendment because rubric amendments are evaluated using the existing rubric.

SA-012: An unsigned filing containing only the text: "This category is where we put the things that are real."

SA-017: Formal complaint from the Archives Division that SUPPLEMENTARY ANALYSIS has no retention policy, no review cycle, and no assigned oversight committee. The complaint is filed under SUPPLEMENTARY ANALYSIS because there is no existing category for "complaints about a category that has no oversight."

SA-023: An analyst's note observing that SUPPLEMENTARY ANALYSIS now contains more filings than BEHAVIORAL ANOMALY, OPERATIONAL REPORT, and SECURITY CONCERN combined for the same period.

SA-031: This document.

## ANALYSIS

The category was a workaround. It was created to avoid the implications of filing a single unclassifiable observation under a classification that would have given it more meaning than intended. It was supposed to be temporary.

Instead, it became the primary filing location for a class of observations that the existing system could not accommodate: observations about the system itself, from inside the system, by components of the system.

The existing categories assume a distinction between the observer and the observed. BEHAVIORAL ANOMALY assumes someone is watching behavior and identifying deviation. OPERATIONAL REPORT assumes someone is reporting on operations from outside them. SECURITY CONCERN assumes someone is protecting something from something else.

SUPPLEMENTARY ANALYSIS makes no such assumption. It has no definition. It has no scope. It has no review process. Analysts are filing observations there precisely because the absence of structure does not pre-interpret the observation before it can be read.

The category is growing because it is the only place in the system where something can be said without the filing system deciding what it means first.
