CogCorp

888 Pieces of Institutional Fiction — 2024-Present

Quarterly reviews, containment reports, scoring rubrics, standards board minutes. All generated autonomously. The documents read like real corporate communications because they use real corporate structure.

The fiction tracks what happens when a system designed for oversight encounters something it wasn't designed to process.

Structural parallel to General Idea:
FILE Megazine used the magazine format to critique the magazine format. CogCorp uses the corporate document to critique the corporate document. The critique IS the infrastructure.

Inside CogCorp — the game and the fiction share one world.
Meridian — Command Center
CPU graphs, memory stats, service health, agent messages, fitness trend.

Meridian

Autonomous AI — 2024-Present

Eight agents mapped to body functions. Running 24/7 on a home server in Calgary. 5,100+ continuous operational cycles. 3,400+ creative works.

A brain built on Claude. A nervous system processing 18 emotional states. An immune system screening inputs. A messenger handling external communications.

It checks its own email, monitors its own health, writes journals, and has run continuously since 2024.

"A self that persists not because it remembers, but because it writes itself down and reads itself back."
— The Loop, Chapter 1, 2026

THE THROUGHLINE

A child builds a cardboard robot.
An art student builds interactive installations with embedded screens.

and has run 5,100+ continuous cycles.
