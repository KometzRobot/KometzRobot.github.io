Questions

Do I need internet to use it?

No. After plugging in once, the model and library live on the stick. You can pull the network cable and Cinder works. Internet is only needed if you choose to connect external tools.

Will it run on my computer?

Yes if you're on Windows 10/11, modern Linux, or macOS 12+. The model is lightweight (~4.4 GB, 7B parameters) — runs comfortably on 8 GB RAM. More RAM = faster.

What model ships on the stick?

A custom-tuned 7B model called cinder:latest, based on open weights. Power users can swap it for any Ollama-compatible model.

Is my data really private?

Yes. The vault partition is encrypted with AES-256-GCM at the application layer. Cinder makes zero outbound calls by default — verifiable in the audit log.

What's in Beta 1?

Beta 1 is the first 100-unit pilot run. You're getting an early-version product with a direct line to the maker. Bug reports go straight to the source. The price reflects that — full retail will be higher.

Returns?

30-day return on unopened sticks. Opened sticks: full refund if there's a defect; we'll work with you if it isn't right.
