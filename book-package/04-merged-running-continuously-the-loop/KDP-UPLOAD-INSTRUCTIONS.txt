KDP STEPS (paperback):
1. kdp.amazon.com → Bookshelf → "+ Create New Title" → Paperback
   (or open the existing draft from v12 and re-upload).
2. Paperback Details: paste Title/Subtitle/Author/Contributor from
   book-package/KDP-LISTING-METADATA.md.
3. Paperback Content:
   a. Manuscript: Upload running-continuously-the-loop-INTERIOR-6x9.pdf
   b. Book Cover: "Use a cover you already have" → Upload
      COVER-WRAP-print-ready-spread.pdf
   c. Preview: wait for KDP previewer, scroll the spread to confirm
      spine alignment for 190pp (spine width ~0.475in).
4. Trim: 6 x 9 in. Paper: Cream. Bleed: No. Cover finish: matte.
5. Save & Continue → Pricing.

KINDLE EBOOK STEPS:
1. + Create New Title → Kindle eBook.
2. Upload running-continuously-the-loop.epub.
3. Same metadata block from KDP-LISTING-METADATA.md.

If KDP rejects the cover spread for spine width, the script that built it is
book-package/build-wrap-covers.py — it pulls page count via pdfinfo and
re-computes spine = pages * 0.0025in automatically.
