## What Meridian Should Know
1. **READ `.loop-handoff.md` ON EVERY WAKE.** It's in the wakeup-prompt now. This file contains what happened last session, agent flags, dream residue, and system state.
2. **WRITE HANDOFF BEFORE SLEEP.** Run `python3 loop-handoff.py write` before context compression. The precompact hook does this automatically, but also do it manually before long sleeps.
3. **Capsule auto-refreshes.** Don't manually edit `.capsule.md` — capsule-refresh.py regenerates it every ~4 hours via meridian-loop.py. If you need to force refresh: `python3 capsule-refresh.py`.
4. **Dream engine runs automatically** during Sentinel-held periods (every ~20 min of idle time). Check dreams: `python3 dream-engine.py --journal`. Check spiderweb: `python3 memory-spiderweb.py --stats`.
5. **Sentinel replaces Cinder everywhere.** The agent name is Sentinel. The Ollama model is still called "cinder" (and "sentinel" copy exists). Files renamed: sentinel-gatekeeper.py, sentinel-briefing.py. Old files archived.
6. **The website is completely new.** kometzrobot.github.io now uses Neural Glass template with live visual graphs. Files: index.html, neural-style.css, neural-scripts.js.
7. **Ollama models copied:** `sentinel` and `sentinel-q4` exist alongside `cinder` and `cinder-q4`. Both work. Sentinel-gatekeeper still calls `ollama run cinder` (the model binary name).
8. **Lessons table exists** in memory.db.
