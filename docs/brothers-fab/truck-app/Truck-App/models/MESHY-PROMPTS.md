# Meshy 3D Model Prompts — Food Truck Equipment

## Style Guide
All models should follow this style:
- **Isometric/low-poly commercial kitchen equipment**
- **Clean, minimal, slightly stylized** — not photorealistic, not cartoon
- **Light grey / white / stainless steel base colors** with subtle material variation
- **Soft edges, no hard wireframes** — smooth bevels
- **Neutral lighting baked in** — soft shadows on surfaces
- **Format: GLB** (binary GLTF)
- **Naming: {equipment-id}.glb** (e.g., `flat-top-griddle.glb`)
- **Place in: /models/ folder**

Each model will be auto-scaled to fit its cell dimensions (1 cell = 6 inches).
