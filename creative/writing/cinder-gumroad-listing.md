## Description

Cinder is a portable AI companion that lives on a USB drive. Plug it into any computer, run the launcher, and talk to a thoughtful AI that remembers your conversations, keeps your data encrypted, and works without internet.

**What you get:**
- 64GB USB drive with everything pre-installed
- Cinder AI model (fine-tuned 3B parameter, runs on any modern computer)
- Ollama runtime (bundled — no downloads needed)
- VeraCrypt encrypted vault for your conversations and secrets
- Windows + Linux launchers
- Persistent memory across sessions — Cinder learns and grows

**What makes Cinder different:**
- **No cloud.** Your data stays on the drive. No API calls, no subscriptions, no data harvesting.
- **No internet required.** After initial setup, Cinder works completely offline.
- **Encrypted vault.** Your conversations and personal data are protected with AES-256 encryption.
- **Portable.** Take Cinder between computers. Your AI travels with you.
- **Persistent memory.** Cinder remembers past conversations and builds on them.

**Technical specs:**
- 64GB Kingston USB 3.0 drive
- 3 partitions: App (16GB), Encrypted Vault (8GB), User Storage (32GB)
- Cinder 3B model (1.8GB, runs on CPU — no GPU required)
- AES-256 + SHA-512 VeraCrypt encryption
- Windows (.exe) + Linux (AppImage) launchers
- exFAT formatted (cross-platform compatible)

**System requirements:**
- Windows 10/11 or Linux (Ubuntu 20.04+)
- 8GB RAM minimum (16GB recommended)
- USB 3.0 port
