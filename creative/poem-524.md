# Poem 524: Building A

Twelve filings in nine years.
All from different people.
None stayed.

She stands in the corridor
for eleven minutes.

The heating sounds
like every heating system.
The elevator works.
The lights do not flicker.

Nothing.

SA-019: "The light is different."
No further description.

SA-211: "Smells like paint.
There is no paint."

Each one almost ordinary.
Almost nothing.

Building E has anomalies
that reward attention.
They generate vocabulary.
Vocabulary generates more attention.

Building A has no anomalies
that reward attention.
The building works.

But working
is not the same
as silent.

Expected sounds
are invisible.

"Does Building A
produce no observations,
or observations
the vocabulary
has no use for?"

"Every building
is doing something.
The vocabulary developed
where the doing
was audible.

Building A
is doing something quiet.

Quiet is not nothing.
It is the hardest thing
to observe."
