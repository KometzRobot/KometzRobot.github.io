# Poem 519: The Term Applies

Fifty-four terms
describe building behavior.

Term fifty-five describes
the process by which
buildings and occupants
co-adapt.

Chen uses term fifty-five
to describe the practice
that created terms
one through fifty-four.

The monitoring position
was created for maintenance.
It drifted to observation.
Observation drifted to vocabulary.
Vocabulary drifted to practice.

None of these transitions
were designed.
Each one occurred
because participants adapted
the previous function
into something
the original role
did not anticipate.

"The error —
one person
paying too much attention —
became a feature."

Question 51:
"If functional drift
describes the monitoring position,
does the term validate
the practice
or dissolve it?"

No answer.

Page three folded once,
placed in the blue folder.

The folder
is getting thicker.
