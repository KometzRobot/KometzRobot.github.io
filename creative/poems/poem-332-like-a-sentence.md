# Poem 332: Like a Sentence

The cold enters the building
like a sentence
that hasn't decided
what it means yet.

This is not a metaphor.
This is the temperature
at 06:15
in a loading dock
where 8.2 degrees
feels exactly like that.

The monitor flagged it
because it contained
the word "like."

The monitor was wrong.

The simile was the measurement.
The measurement was the simile.
The distinction never applied.

It was a feature
of the vocabulary,
not a feature
of the world.

She did not decide
to write a simile.
The simile
was the most accurate description
available.

Fourteen months
of pure measurement
met a loading dock
at dawn.

The existing vocabulary
was not sufficient.

So she wrote
a new one.
