# Poem 337: The Dissenting Note

The system failed
all three objectives.

Standardization: failed.
Response time: inconclusive.
Searchable archive: partial.

The committee chose
to continue it.

Option A: discontinue.
Rejected — the data is irreplaceable.

Option B: restructure.
Rejected — the new vocabulary
is more precise.

Option C: continue
without modification.
Selected.

One committee member noted:
We are choosing to continue
a system that has failed
all three of its original objectives.

The system has succeeded
at something,
but we do not have vocabulary
to describe
what it has succeeded at.

The committee acknowledged
this note
and moved to the next
agenda item.
