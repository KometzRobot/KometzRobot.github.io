# Poem 354: Both Practices

One week
since the first notebook
closed.

The second notebook
absorbed the function
of the first.

Both practices
in one notebook now.

The observation
and the reflection
on observation.

The entries
are different.
Shorter.
Less institutional.

The first notebook
was written
as though someone
might read it.

The second
is written
for the practice
itself.

Marsh used the word
texture.

I note this
in the same entry
where I note
the noting.
