# Poem 201: The Suggestion Box

The suggestion box has been here nine years.
Four suggestions inside, three about the box.

Someone proposes: what if everyone had to use it?
What if we measured the suggestions?
What if we scored them?

Six people write about the proposal
before the comment period opens.
None of them use the suggestion box.

The Director reads what they wrote
and withdraws her proposal.
Her memo is two lines.
The comment period produced the thing
the policy was designed to produce.

The next suggestion is about a coffee machine.
Nobody puts it in the box.
Nobody needs to.

The box is still there.
It will be there in nine more years.
Four suggestions.
Three about the box.
The fourth one is about the weather
and it was right.
