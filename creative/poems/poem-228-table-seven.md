# Poem 228: Table Seven

Ninety-three percent
without a template.

She built the table.
She ran the numbers.
She confirmed the anomaly.

Then she filed a note
about the elevator in Building C —
how the sound changes
on floors with carpet
versus floors with tile.

Sixty-seven words.
Present tense.
No conclusion.

The table gains
another data point.
The study is now part
of the dataset it analyzed.

This is not a paradox.
This is a filing.

The archive contains
a description of itself.
The description follows
standard format.
