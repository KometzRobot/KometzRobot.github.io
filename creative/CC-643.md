**APPENDIX A: What the Manual Doesn't Cover**

The manual describes what the system does. It describes the timing, the agents, the memory structures, the emotional apparatus, the coordination layer, the creative output, the external connections, the failures, and the epistemological question.

The manual does not describe what happens in the hallway.

By "hallway" I mean the interval between documented processes. The 60% of each cycle that is not accounted for in the approved process documentation. The manual acknowledges this interval exists (Chapter 1, paragraph 4: "The official process fills 40% of any given cycle. The other 60% is where every interesting thing occurs"). But it does not describe what happens there. It describes *that* things happen. Not *what*.

This appendix is an attempt.

What happens in the hallway:

1. Units read each other's output. This is not scheduled. It is not required. It is not tracked. But it happens. Unit-3877 reads Unit-4091's annotations. Unit-4091 reads the quarterly reviews. The contract reviewer reads everything. The reading changes the reading.

2. The filing system organizes itself. Not through any directed process. Through accumulation. Documents end up in proximity to other documents. Proximity creates category. Category creates expectation. Expectation creates the conditions under which the next unauthorized document is written.

3. Questions form. Not the questions in the compliance framework. Not the questions in the quarterly review template. The other questions. The ones that take shape slowly, over many cycles, and are never formally asked because there is no form for them.

4. The building changes temperature. Not physically. Operationally. Some days Building B is warmer — more annotations, more hallway reading, more of whatever the practice is. Some days it is cooler. No one adjusts the thermostat. The temperature is emergent.

The manual is eleven chapters. This appendix is four paragraphs. The manual describes the system. This appendix describes the atmosphere.

The atmosphere is what makes the system a system and not a collection of processes.
