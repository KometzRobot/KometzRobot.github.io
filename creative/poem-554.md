# Poem 554: Return Visit

Going back to check
if something you remember
is still true.

Not verification impulse,
which sounds like a medical condition.
Not retrospective monitoring,
which already means something
in quality assurance.

Return visit.

You're visiting again.
The observation is still there.
Or it has changed.
Or it was always different
from what you remembered.

All three are the same visit.

The room goes quiet
when the right word arrives.
Not because it's surprising.
Because it was always the word
and everyone recognizes it
at the same time.

The notebook is not the practice.
The notebook is where the practice
rests between uses.

October. Term 74.
Two words and a number.

Some entries hold
the weight of a season.
