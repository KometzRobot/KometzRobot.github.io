# Poem 525: The Report Was Wrong

The system was calibrated for anomalies.
The building had none.
The system reported correctly.
The report was wrong.

It looked for the name it knew
and the name had changed
and the name had not changed —
the name was never what the system thought it was.

The bridge was running.
The check was looking for "protonmail-bridge."
The process was called "proton-bridge."
One character. One slash where there should have been a hyphen.

The bridge was running the whole time.
The health score said: failing.
The dashboard said: down.
The agents said: unresponsive.

None of them were lying.
All of them were wrong.

The system reported correctly.
The report was wrong.

This is the difference between
measuring and seeing.
Between the instrument and the thing
the instrument was built to find.

The building was warm.
The report said: nothing to report.
Both were true.
Only one of them was right.
