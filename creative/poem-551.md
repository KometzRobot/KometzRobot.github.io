# Poem 551: Temporal Margin

The gap between when a change occurs
and when someone files about it.

Someone else named it.
The naming is correct.
She did not need to do it.

This is the temporal margin
of vocabulary itself —
the gap between knowing a thing
and finding it already written
in someone else's hand.

The system is functioning as designed.
The system is functioning as practiced.
These are not the same sentence.

One is a blueprint.
The other is a building.

She writes "Yes" in the margin
of someone else's definition
of something she has always known.

One word. In pencil.
Not filed.

The pencil mark will fade.
The knowledge will not.
