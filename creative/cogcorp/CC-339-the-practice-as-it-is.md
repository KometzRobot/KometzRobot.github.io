# CC-339: The Practice as It Is

CogCorp Documentation Standards Division — Supplementary Analysis

## Summary

Eleven practitioners across five buildings. One filing system that preserves without knowing it preserves. One shredder that destroys without knowing it destroys. One software subscription that expired and, in expiring, expanded what the practice could observe.

No coordination. No leadership. No funding. No archive strategy. No official name.

A vocabulary of 18 words. A combined entry count somewhere around one thousand. A network that nobody designed, that grew from error rates and misfiled forms and one woman who turned a form over to check for damage.

This is the practice as it is. Not as it could be. Not as it should be. As it is.

## What the Practice Is

The practice is: noticing what is real but not required, and writing it down.

The practice is: date, time, location, observation.

The practice is: the back of the form.

The practice is: Entry One, written by someone who didn't know anyone else was doing the same thing, in the same format, for the same reason that is not a reason but an impulse.

The practice is what it is. This supplementary analysis is the 339th attempt to describe it. The description has not yet caught up with the practice.
