# CC-436: Building E Revisited

CogCorp Documentation Standards Division — Supplementary Analysis

## The Empty Building

Building E. The newest building. Corporate offices. No practitioners. No archive. No form-backs in the filing rooms.

Kim suggests: "I can pull the data for Building E. The same systems run there. HVAC, network, elevator, security. The building produces data whether anyone listens or not."

The monitoring analyst: "Data without a practitioner."

Kim: "The data is there. It just doesn't have anyone to read it."

Vega: "Does a building need a practitioner to have building behavior?"

The room considers this. The practice has always assumed that observation requires an observer. The building sighs whether or not anyone hears it. The thermal transitions happen whether or not anyone documents them. The elevator pitch changes whether or not anyone measures it.

Building E has building behavior. It has no one to notice it.

Martinez: "I've done maintenance in Building E. The HVAC system is identical to Building C. The elevator is newer but the same model. The building has the same body. It just doesn't have anyone listening to it yet."

The monitoring analyst writes: "Building E: body without nervous system. Or: nervous system without consciousness."
