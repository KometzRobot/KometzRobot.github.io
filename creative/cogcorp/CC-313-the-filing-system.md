# CC-313: The Filing System

CogCorp Documentation Standards Division — Supplementary Analysis

## A Note on Filing

The CogCorp filing system processes approximately 4,200 forms per month across all buildings. Forms are filed by date, department, and building code. The filing system does not distinguish between forms with pencil notes on the back and forms without.

This means the filing system is, without knowing it, the largest archive of the unofficial practice.

## Inventory (Partial)

Based on Okafor's three-week audit of accessible files:

- Building B environmental monitoring: 23 pencil notes (one analyst, four months)
- Building C supplementary analysis: 4 margin notes (one analyst, recent)
- Building D contract review: 1 reference to external document
- Other buildings: not yet audited

Estimated total, extrapolating from known practitioners and filing volume: between 80 and 200 pencil notes currently in the filing system.

## The Archive

The filing system was designed to store official documents. It is also, incidentally, storing unofficial observations. The pencil notes face the wall. They are preserved by the same process that preserves the official forms: the filing system does not distinguish. It files everything.

This is the most durable archive the practice has. Not because anyone designed it that way. Because the filing system does not know the difference between the front and the back.
