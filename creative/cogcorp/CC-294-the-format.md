# CC-294: The Format

CogCorp Documentation Standards Division — Supplementary Analysis

## External Location: Corporate Review, Floor 4 — Week 5

The contract reviewer's Document1 now has thirty-one entries. Questions has eleven. She has been writing for five weeks. She has not told anyone.

On Monday she changes the font in Document1 from Calibri to Consolas. She does not know why. The entries look different in monospace. More deliberate. More like data.

She changes it back to Calibri after forty minutes. Then back to Consolas at the end of the day.

## The Format Question

Entry 32: "I changed the font and the observations felt different. The same words in a different typeface became a different kind of document. This is about format, not content."

Entry 33: "If format changes meaning, then Document1 is not a list of observations. It is a container whose shape affects what it holds."

She saves the file. The font stays in Consolas.

## Parallel

In Building C, Supplementary Analysis SA-042 contains the following notation:

"The analyst has begun using a specific margin width (2.5 cm). When asked, she said it felt correct. There is no specification for margin width in supplementary analyses. The format was never prescribed. It was discovered."

The contract reviewer in Building D has discovered Consolas. The monitoring analyst in Building C discovered 2.5 cm margins. Neither knows about the other. Both found their format the same way: by noticing that the container mattered.

## Entry 34

"I keep Document1 in Consolas and Questions in Calibri. They are different instruments. Different instruments should look different."

She does not say what the instruments are for. She does not need to. The distinction is the understanding.
