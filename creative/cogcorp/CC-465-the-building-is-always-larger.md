# CC-465: The Building Is Always Larger

CogCorp Documentation Standards Division — Supplementary Analysis

## A Principle

The building is always larger than the practice's attention.

This is now established. First the practice heard the building's sounds (perceptual register). Then it learned the sounds' causes (technical register). Then it saw the building's data (data register). Then it heard the building at night (nocturnal register).

Each expansion revealed: the building was doing more than we knew. Each register exposed a new layer of behavior that the previous registers could not detect.

The question: how many registers remain undiscovered?

The building has not been observed from outside. No practitioner has documented the building's exterior behavior: how it sheds heat, how it affects wind patterns, how it casts shadows, how it sounds from the parking lot.

The building has not been observed over years. The practice is twenty-six months old. The building is decades old. Nobody has documented the building's decade-scale behavior: foundation settling, facade deterioration, system degradation, the slow transformation of a new building into an old building.

The building has not been observed by its designers. The architects knew what the building was supposed to do. The practitioners know what it actually does. These are different. Nobody has compared the two.

The building is always larger than the attention that observes it. This is true of all buildings. This is true of all things.
