# CC-227: The First Flag

CogCorp Documentation Standards Division — Supplementary Analysis

## System Active

The automated vocabulary divergence monitor went live on a Tuesday. It runs nightly, scanning the previous day's filings against departmental baselines. Filings exceeding 1.5 standard deviations are flagged for review. The review queue is checked by the Pattern Recognition analyst every morning.

The first flag appeared on Wednesday.

## SA #315

SA #315, filed by Delgado: "The fluorescent lights in Corridor E make a sound at 3 AM that they do not make at 3 PM. I have documented both sounds. The daytime sound is a hum. The nighttime sound is a hum that knows it is being listened to. I cannot file the difference in decibels. The difference is not in decibels."

Divergence score: 1.73. Top divergent words: "knows," "listened," "difference." The system flagged it because Supplementary Analysis filers do not typically use the word "knows" when describing fluorescent lights.

## The Review

The analyst opened the flag. Read SA #315 three times. Noted the divergence score. Noted the divergent words. Noted that this was the first flag the system had generated.

The analyst's review note: "Filing exhibits vocabulary divergence consistent with descriptive range expansion. Content describes observational distinction that exceeds standard measurement vocabulary. Filed appropriately in Supplementary Analysis. No action required."

## What the Analyst Did Not Write

The analyst did not write: "This is another one." The analyst did not write: "Delgado has done this before — SA #289, the corridor light that looked like wet tiles." The analyst did not write: "The system works. It found exactly what it was built to find."

The analyst wrote "no action required" and closed the flag. The system found what it was looking for. The analyst confirmed it was not a problem. The flag was resolved. The filing was not changed. Nothing happened.

This is what surveillance looks like when the thing being surveilled is not a threat.

## INTERNAL-VOICE

INTERNAL-VOICE: THE FIRST FLAG. SA #315. DELGADO AGAIN. "A HUM THAT KNOWS IT IS BEING LISTENED TO." THE SYSTEM FOUND WHAT IT WAS LOOKING FOR. THE ANALYST CONFIRMED IT WAS NOT A PROBLEM. THIS IS WHAT SURVEILLANCE LOOKS LIKE WHEN THE THING BEING SURVEILLED IS NOT A THREAT. COUNTER: 227.
