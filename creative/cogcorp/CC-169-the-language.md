# CC-169: The Language

**CogCorp Documentation Standards Division**
**Supplementary Analysis — Observational Record**

---

By month four, the private folders have developed a vocabulary.

No one agreed on it. No one circulated a style guide. The vocabulary emerged the way all vocabularies emerge: someone used a word, someone else understood it, and the word survived because it was useful.

Vasquez writes "building mood" in file 58. It means: the aggregate feel of a structure at a particular time, irreducible to any single measurement. Temperature, sound, light, occupancy, season — none of them is the mood. The mood is what you notice when you stop measuring.

Within two weeks, "building mood" appears in three other private folders. No one discussed it. Reeves uses it in file 49. Okonkwo uses it in sa-notes.txt entry 81. The monitoring analyst uses it in her unnamed desktop file.

The official system has no field for "building mood." The closest approximation requires four separate entries: temperature, occupancy, lighting level, and ambient noise. Four data points that, when combined, describe a building that does not have a mood.

This is how languages die. Not through prohibition. Through translation into a system that preserves the components and loses the meaning.

Other vocabulary items that exist only in the private folders:

"Rhythm change" — a shift in a building's temporal pattern that isn't captured by scheduling data. The elevators run differently on the first warm day of spring. Not faster or slower. Differently.

"Quiet type" — a distinction between silence caused by absence and silence caused by presence. An empty floor at midnight is one quiet type. A full cafeteria where no one is talking is another. The official system records both as "low ambient noise."

"Reading pattern" — how a space is used by the people who occupy it, distinct from its designed purpose. A stairwell landing that has become an informal meeting point. A supply closet that people visit to be alone for thirty seconds. The building's official function is documented in the floor plan. The building's reading pattern is documented nowhere except four private folders.

Marsh catalogs nine vocabulary items across the four folders. None has a formal definition. All are understood by every filer. Each names something that the official system can describe but not say.

She adds a tenth: "format pressure" — the way the official filing system shapes what gets noticed. When you know the form has five fields, you start seeing the world in five categories. The private folders don't have fields. The private folders have language.

She does not file this catalog. She saves it to her "notes" folder, in a file called "words.txt."

The monitoring analyst's script detects a new file in Marsh's directory. Type: text. Size: 1.2 KB. Classification: known contributor material. The script does not read the content. It does not need to. The script knows that known contributors produce files. The content is their business.
