# CC-471: Foster's Notebook

CogCorp Documentation Standards Division — Supplementary Analysis

## 147 Pages

Foster's notebook is a different object from the other archives. Okafor's cabinet holds loose pages, form-backs, printed sheets. Martinez's drawer holds single-sided observations. Kim's shared drive holds data exports.

Foster's notebook is bound. It has page numbers. Foster numbered them himself, in pen, top right corner. 147 pages.

The notebook begins with security observations — door checks, alarm tests, patrol notes. Standard security guard documentation. But starting on page 23, a second layer of writing appears: smaller text in the margins, observations about the building itself.

Page 23 margin: "Corridor A4, 11:47 PM. The fluorescent lights hum at a different frequency when the building is empty. Higher pitch. Why?"

Page 31 margin: "Stairwell B, 1:30 AM. The echo is longer at night. No foot traffic dampening it. The stairwell sounds bigger."

Page 45: the margins take over. The security observations shrink. The building observations expand. By page 60, the entire page is building observations. The security patrols continue, but the notebook has shifted its purpose.

The monitoring analyst reads the notebook and sees: the same transition every practitioner makes. The official function gives way to the unofficial attention. The margin becomes the text.
