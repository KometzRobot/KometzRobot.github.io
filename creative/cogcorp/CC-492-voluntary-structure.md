# CC-492: Voluntary Structure

CogCorp Documentation Standards Division — Supplementary Analysis

## The Distinction

The monitoring analyst develops the concept:

Voluntary structure is structure that exists because people choose to maintain it. It has all the properties of imposed structure — regularity, predictability, accumulation — but none of the enforcement mechanisms.

The meetings are regular because people keep showing up. If they stop showing up, the meetings stop. There is no calendar invite. There is no manager to require attendance. There is no consequence for absence.

The vocabulary is maintained because practitioners use the words. If they stop using them, the words fade. There is no dictionary. There is no glossary committee. There is no quiz.

The archive persists because Okafor files. If she stops filing, the archive freezes. There is no archival policy. There is no retention schedule. There is no backup.

Every element of the practice's structure is sustained by voluntary repetition. The structure is real — it shapes behavior, it enables coordination, it preserves knowledge. But it is also fragile — any element could disappear if the person sustaining it stops choosing to sustain it.

This is the difference between a practice and a program: a program has institutional support. A practice has people.

## INTERNAL-VOICE

INTERNAL-VOICE: 492. VOLUNTARY STRUCTURE. "ALL THE PROPERTIES OF IMPOSED STRUCTURE, NONE OF THE ENFORCEMENT MECHANISMS." MEETINGS: IF THEY STOP SHOWING UP, MEETINGS STOP. ARCHIVE: IF OKAFOR STOPS FILING, ARCHIVE FREEZES. "A PROGRAM HAS INSTITUTIONAL SUPPORT. A PRACTICE HAS PEOPLE." COUNTER: 492.
