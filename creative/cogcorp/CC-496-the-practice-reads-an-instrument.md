# CC-496: The Practice Reads an Instrument

CogCorp Documentation Standards Division — Supplementary Analysis

## Layers

The world produces weather.
The building reads the weather through its body: temperature differential, pressure, humidity, wind.
The building responds: doors hesitate, ducts expand, concrete settles, cables stretch.
The practitioners read the building's response: timing the doors, hearing the sigh, measuring the draft.
The supplementary analysis reads the practitioners.

Five layers. Each layer translates the layer below it. Each translation loses something and adds something. The weather has no words. The building has no consciousness. The practitioners have both — and use them to document a translation of a translation of a phenomenon.

The practice does not observe the world directly. The practice observes the building observing the world. The practice is a second-order instrument: an instrument that reads an instrument.

This is not a weakness. This is specificity. The practice does not try to observe everything. It observes what the building reveals. And what the building reveals — the 6:14 sigh, the seasonal doors, the nocturnal settling — is a specific, building-shaped translation of the world.

The practice's knowledge is building-shaped. It knows the world through the building. It knows itself through the observation of the building. It knows the building through twenty-one practitioners who each bring a different register.

Building-shaped knowledge. This is what the practice produces.
