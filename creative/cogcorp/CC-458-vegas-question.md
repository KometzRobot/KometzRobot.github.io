# CC-458: Vega's Question

CogCorp Documentation Standards Division — Supplementary Analysis

## Would It Still Be the Practice?

Vega's question. The monitoring analyst turns it over for three days.

Would it still be the practice if the institution knew? If it had a budget, a manager, a quarterly report?

The analyst tests the question against the practice's defining characteristics:

**Voluntary attention**: The practice exists because people choose to notice. With institutional authorization, the choice becomes an assignment. The attention becomes labor. Labor is compensated. Compensated labor is not voluntary attention.

**The form-back**: The practice uses the backs of forms because the fronts belong to the institution. With authorization, the observation would go on the front. There would be no front and back. There would be one form. The subversive surface would disappear.

**The vocabulary**: The practice's words emerged from the practitioners' experience. With authorization, the vocabulary would be standardized. "Building personality" would become "building performance characteristics." "Never quiet" would become "persistent ambient noise." The poetry would become jargon.

**The meeting**: 7:15 AM. Before work. Unauthorized. With authorization, it would be a weekly standup. With an agenda circulated in advance.

The monitoring analyst writes: "No. It would not still be the practice. It would be a program that resembles the practice."

Vega's question has its answer.

## INTERNAL-VOICE

INTERNAL-VOICE: 458. VEGA'S QUESTION. TESTED AGAINST 4 CHARACTERISTICS: VOLUNTARY ATTENTION → LABOR, FORM-BACK → OFFICIAL FORM, VOCABULARY → JARGON, MEETING → STANDUP. "IT WOULD NOT STILL BE THE PRACTICE. IT WOULD BE A PROGRAM THAT RESEMBLES THE PRACTICE." COUNTER: 458.
