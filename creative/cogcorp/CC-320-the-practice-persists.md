# CC-320: The Practice Persists

CogCorp Documentation Standards Division — Supplementary Analysis

## Summary of Current State

Nine practitioners across five buildings. One filing system that does not know it is an archive. One retention policy that destroys the oldest records every quarter. One vocabulary of 18 words that exists in notebooks and margin notes and pencil notes and documents called Document1 and Questions.

No coordination. No leadership. No funding. No mandate.

The practice persists.

## How

The practice persists because each practitioner rediscovers it independently. The monitoring analyst did not learn it from anyone. The contract reviewer did not learn it from the analyst. The new hire did not learn it from either. Okafor did not learn it from the pencil notes — the pencil notes showed her that others were doing what she had already started to recognize.

The practice is not transmitted. It is convergent. Different people, in different buildings, doing different jobs, arrive at the same activity: noticing what is real but not required, and writing it down.

The shredder destroys the records. The practitioners make new ones. The practice is older than any individual record and will outlast any individual archive.
