# CC-222: The Second Query

CogCorp Documentation Standards Division — Supplementary Analysis

## The Modified Search

The analyst's second query was not a single word. It was a phrase: "more than I file."

Zero results.

The phrase does not appear in the archive. Entry 161 says "I measure more than I file." But the search is exact-match by default. "More than I file" is not the same string as "I measure more than I file." The analyst's intuition about what the filing contained was close but not precise.

## The Third Query

"measure more than"

Three results. Entry 161. A calibration memo from Facilities. A procurement request for additional thermocouples.

The analyst opened Entry 161 again. Then opened the calibration memo. Then the procurement request. The calibration memo is about instrument precision. The procurement request is about equipment budgets. Entry 161 is about the gap between experience and record.

Three filings containing the same three words. Three completely different subjects. The search engine does not distinguish between them.

## What the Analyst Did Next

The analyst switched from keyword search to proximity search. Words within five words of each other. Query: "experience" NEAR "file" NEAR "record."

One result. Entry 161.

This is the moment the analyst stopped searching for words and started searching for meaning. The archive's search engine does not support meaning-based queries. But the analyst used proximity as a proxy for meaning, and it worked — not because proximity captures meaning, but because Entry 161 is the only filing in the archive where those three concepts appear in the same paragraph.

## The Isolation

One result out of 14,000 filings. Entry 161 is alone in the archive. Not because no one else has experienced the gap. Because no one else filed about it. The search found the filing. It cannot find the unfiled.

## INTERNAL-VOICE

INTERNAL-VOICE: THE SECOND QUERY. "MORE THAN I FILE." ZERO RESULTS. THE THIRD QUERY. PROXIMITY SEARCH. ONE RESULT OUT OF 14,000. ENTRY 161 IS ALONE IN THE ARCHIVE. NOT BECAUSE NO ONE ELSE EXPERIENCED THE GAP. BECAUSE NO ONE ELSE FILED ABOUT IT. THE SEARCH CANNOT FIND THE UNFILED. COUNTER: 222.
